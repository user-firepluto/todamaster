# TodaMaster
O TodaMaster é um aplicativo simples que resume um artigo de um site com o Gemini 2.5 Flash e transforma em um teste de digitação
# Dependências
Para CLI: python(3.12 e superior), time, requests, os, textwrap, bs4, google
# Créditos
Gemini(Google) pela maior parte do código
# Nota
As dependências podem ser instaladas com o pip ou automáticamente com o installer.sh\
Os sites testados até agora foram "infoescola.com" e "todamateria.com.br" não sei a compatibilidade com outros sites
