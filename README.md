# TodaMaster
O TodaMaster é um aplicativo simples que resume uma página do site "todamateria.com.br" com o Gemini 2.5 Flash e transforma em um teste de digitação
# Dependências
Para CLI: python(3.12 e superior), time, requests, os, textwrap, bs4, google
Para GUI: python(3.12 e superior), time, requests, webbrowser, bs4, google, PySide6
# Nota
As dependências podem ser instaladas com o pip
